# events-hospitality.md — Events & Hospitality Vertical Template
## Hotel · Event Venue · Wedding Venue · Catering · Photographer · Event Planner

**Applies to:** Events & Hospitality clients across product types 1–3 (per `TECH.md` §1). Hotel chains and large venues are Type 3 (booking-DB + availability calendars). Single event venues + boutique hotels are Type 2 (inquiry forms). Solo photographers/planners are Type 1–2. The visual product across all sub-verticals is **atmosphere** — gallery-first is non-negotiable.

**Reference research:** Based on `local_business_website_benchmark_report.md` §7.12 (Fasano, Hotel Unique, Casa Petra, Grupo Bisutti, Buffet França, Marriott, Four Seasons, Arlo, Great Performances, Wolfgang Puck Catering, Accor, The Hoxton, Soho House, citizenM, Hedsor House).

**Use this doc as a moodboard with principles, not a layout to clone.** If two clients in the same vertical end up with the same hero pattern, the template has failed at its job.

---

## Rules at a glance

- **The atmosphere IS the product.** Photography and gallery sequencing are the conversion path, not service descriptions. A venue page without 20+ professional photos fails before the first scroll.
- **Pick one of four archetypes** before designing: **Luxury Hospitality** (Fasano, Four Seasons), **Boutique Hotel/Brand-Led** (Hoxton, citizenM, Arlo), **Event/Wedding Venue** (Hedsor House, Casa Petra), or **Solo Photographer / Event Planner / Caterer**. ~60 % of agency clients fall into Solo or single-venue Event.
- **The conversion is inquiry, not booking** for single venues + solo operators. Wedding venues, event planners, photographers, caterers all convert through inquiry forms (date + headcount + budget). Hotel chains convert through availability calendars; that's a different product type entirely.
- **Capacity + packages + pricing-range** must be visible early for venues. "Up to 200 guests · packages from €5,000" beats "Contact for pricing."
- **Real venue photography** is mandatory. Stock event photos (champagne toast / candle-lit table) are the loudest tell.
- **Video / virtual tour** is high-leverage for venues + hotels. Atmosphere transmits better via motion than stills.
- **Avoid script fonts and rose-gold-everywhere wedding-blog aesthetics** unless the venue's actual brand is that direction.

### Solo-Operator meta-archetype (cross-vertical pattern)

Most agency Events clients (solo wedding photographer, single event planner, family caterer) are the Events-specific implementation of the **Solo-Operator meta-archetype** in `DESIGN-BEST-PRACTICES.md` §3:

- Operator/business name in headline ("Sara Mendes — Fotografia de Casamento em Niterói")
- Portrait of the operator + signature work
- ONE primary CTA (inquire about your date / WhatsApp / consultation request)
- Credentials signature line (years in business + style + service area + specialties)
- Service list (4–6 packages or styles, not a card grid)
- Service area named explicitly
- Pricing reassurance (starting-from price or consultation flow)

### Sourcing rules (apply before any visual decision)

- **Photo + favicon sourcing:** `DESIGN-BEST-PRACTICES.md` §3 — 8-tier photo + 5-tier favicon priority. **Events-specific:** existing client website (tier 2) is the most common source — most established venues/photographers have galleries already. Booking platforms (TripAdvisor, Booking.com for hotels; The Knot, Casamentos.com.br, Bridebook for weddings) are tier 3 alternatives. Instagram is tier 4 — photographers/planners post daily curated work.
- **Color palette sourcing:** `DESIGN-BEST-PRACTICES.md` §5 — 6-tier color source hierarchy. Premium hospitality often has formal brand guides (tier 1); boutique hotels often have a strong identity in existing site (tier 2); solo operators usually fall to vertical-default (tier 5).
- **Prospect intake template:** `CHECKLIST.md` §9.

## 1. The four Events & Hospitality archetypes

| Archetype | Brand priority | Primary CTA | Production cost |
|-----------|---------------|-------------|-----------------|
| **A. Luxury Hospitality** (Fasano, Four Seasons, Soho House) | Aspirational atmosphere, exclusivity, destination storytelling | Reserve / Become a member | Highest — multi-property booking + editorial photography + global locations |
| **B. Boutique Hotel / Brand-Led** (Hoxton, citizenM, Arlo, Hotel Unique) | Brand personality, neighborhood culture, design-led identity | Book a room / Explore neighborhoods | High — distinctive brand voice + property pages + booking engine |
| **C. Event/Wedding Venue** (Hedsor House, Casa Petra, Grupo Bisutti) | Atmosphere + capacity + inquiry flow | Inquire about your date | Medium — gallery-first + inquiry form + package/capacity tables |
| **D. Solo Photographer / Event Planner / Caterer** (the agency's default Events client) | Operator portfolio + style + service area | Inquire / Request quote | Low — Tier 2 build, portfolio-led |

**Default for new Type 1/2 clients:** Archetype D. Archetype C if the client owns a venue.

## 2. Information architecture per archetype

### Archetype A — Luxury Hospitality

1. **Hero** — large editorial photo of the most iconic property/location + brand wordmark + ONE CTA ("Find your stay" or "Reserve").
2. **Destinations / Properties** — each location as an editorial entry (photo + name + brief evocative copy + "Discover →"). NOT a card grid.
3. **Loyalty program** — Fasano · Four Seasons Premier · Soho House membership.
4. **Editorial content** — magazine-style stories about destinations, chefs, design.
5. **Booking engine** — date pickers + property selector + room types.
6. **Footer** — heavy: properties index, careers, brand standards, sustainability.

### Archetype B — Boutique Hotel / Brand-Led

1. **Hero** — distinctive brand photo (often quirky / culture-led) + headline reflecting brand voice + ONE CTA ("Book a stay" or "Find your Hoxton").
2. **Property locator** — neighborhoods/cities as cards or editorial entries. Each property has its own personality.
3. **Neighborhood storytelling** — local guides, restaurant recs, cultural picks tied to the property.
4. **Rooms / dining / events** — main service categories.
5. **Brand promise** — what makes this hotel chain distinctive (Hoxton's local culture, citizenM's affordable design, Arlo's experiential).
6. **Booking engine** — typically lighter than A — fewer room types, more streamlined.
7. **Footer** — properties index, careers, terms, brand-personality maintained.

### Archetype C — Event/Wedding Venue

1. **Hero** — stunning venue photo (interior shot of event-ready space OR aerial drone of grounds) + headline + ONE CTA ("Inquire about your date").
2. **Gallery** — large grid of 20+ photos covering: venue exteriors, interiors, multiple event setups (wedding / corporate / private), seasonal looks, real-event candids.
3. **Capacity + packages** — table of capacities per space (Ballroom 200 / Garden 150 / Library 60) + package tiers (Standard / Premium / All-Inclusive) with starting prices.
4. **Real events / portfolio** — recent weddings + corporate events as case-study mini-pages.
5. **Venue features** — kitchen / parking / accessibility / catering / outdoor space.
6. **Inquiry form** — date + headcount + event type + budget + contact.
7. **Reviews / press** — wedding-magazine features, real-couple testimonials.
8. **Footer** — Impressum/legal + accreditation + careers.

### Archetype D — Solo Photographer / Event Planner / Caterer (the agency's default)

1. **Top bar** — phone + WhatsApp + (photographers) Instagram link prominent.
2. **Hero** — operator's signature work (photographer's best shot / planner's best event / caterer's best dish-spread) + headline naming operator + specialty + service area + ONE CTA ("Solicitar orçamento" / "Check my availability").
3. **Style/portfolio overview** — 3-section editorial intro: a defining recent event + style explainer + signature element.
4. **Service packages** — 3–5 package tiers OR "Custom proposal based on your event" if everything is bespoke.
5. **Portfolio gallery** — large grid of past work, organized by event type or chronology.
6. **About / Bio** — operator photo + short bio + credentials/experience.
7. **Reviews / testimonials** — verbatim couple/host quotes + small client logo strip if applicable.
8. **Process** — "What working together looks like" — 4-step walkthrough.
9. **Inquiry form** — date + event type + service desired + brief note.
10. **Footer** — Impressum/Política/legal + Instagram + association memberships (e.g., PPA for photographers).

## 3. Hero patterns

### Archetype A — Luxury hospitality

- Large editorial photo of iconic location (Fasano Boa Vista pool, Four Seasons signature property).
- Headline often a destination + tagline ("Where Brazil meets Italy" for Fasano).
- ONE CTA: "Reserve" or "Find your stay."
- Restraint over volume.

### Archetype B — Boutique hotel brand

- Photo capturing brand personality (Hoxton's quirky-neighborhood feel, citizenM's playful-design moment).
- Headline reflecting brand voice ("Your home away from home, with better drinks").
- ONE CTA: "Book a stay" or "Find your Hoxton."

### Archetype C — Event/wedding venue

- The most stunning interior or aerial of the property + ONE CTA ("Inquire about your date").
- Headline: venue name + experience ("Hedsor House — your wedding, your way").
- Date / inquiry form below the fold, never above.

### Archetype D — Solo photographer/planner/caterer

- Signature work as the hero photo.
- Headline names operator + specialty + service area ("Sara Mendes — Wedding Photography in Niterói and Rio").
- ONE CTA: "Solicitar orçamento" or "Check my availability."
- Credentials signature line below (years + style + featured publications).

## 4. Photography direction

### Universal Events & Hospitality rules

- **Real venue, real events, real people** (consent-cleared for any faces).
- **Multiple seasons + setups** for venues. A venue that looks great only in May fails to convert November weddings.
- **Hero photo is the conversion.** It's the single most-leverage image on the site.
- **Detail shots matter.** Place settings · architectural details · candle moments · food close-ups. Atmosphere is in the details.
- **Avoid the cheesy stock kit.** Champagne-glasses-toasting / bride's-shoes-on-windowsill / "dance-floor blur" — universally overused.
- **Drone/aerial for venues** is now table-stakes for ≥ 1000 m² properties.

### Per-archetype photography notes

| Archetype | Photo style |
|---|---|
| A — Luxury hospitality | Editorial-grade commissioned shoots · cuisine, suites, spa, signature views · multi-property consistency |
| B — Boutique hotel | Brand-character shoots · neighborhood context · room personality · breakfast-bar/bar atmosphere |
| C — Event/wedding venue | Venue empty (architectural beauty) + venue dressed (event-ready) + venue in real events (consent-cleared) · multiple setups across same space |
| D — Solo photographer/planner/caterer | The operator's *own work* IS the photography — their portfolio. Operator portrait + 20–50 portfolio images. |

## 5. Typography pairings

### What works in Events & Hospitality

| Pairing | Display | Body | Mood |
|---|---|---|---|
| **Luxury heritage** | Tiempos Headline · Reckless · Lyon Display | Inter / Söhne | Fasano / Four Seasons / Soho House register |
| **Boutique brand-led** | varies wildly with brand identity (Hoxton uses Sentinel, citizenM uses GT America) | Inter / Söhne | Boutique brand voice — match the actual brand |
| **Wedding-venue editorial** | Cormorant Garamond · Fraunces · Spectral | Inter / Manrope | Wedding venues — feminine-leaning premium without rose-gold cliché |
| **Solo photographer** | Cormorant · Lyon · Fraunces | Manrope · Inter | Editorial-portfolio register |
| **Solo planner/caterer** | Fraunces · Cormorant | Manrope | Warmer service-led register |

### Rules

- **Never script-typeface "Mr. & Mrs."** logos. Wedding-blog cliché.
- **Never Comic Sans, never Curlz.** Universally amateur.
- **For venues: ONE display font, ONE body font.** Variety in events comes from photography, not type.

## 6. Color archetypes

### Archetype A — Luxury Hospitality

| Direction | Palette | Mood |
|-----------|---------|------|
| **Heritage cream** | Cream bg / charcoal text / muted brass accent | Fasano / Four Seasons restraint |
| **Soho House dark** | Near-black bg / cream text / single brass accent | Members-club exclusivity |
| **Coastal premium** | Off-white bg / dark navy text / warm gold accent | Beach / island luxury |

### Archetype B — Boutique Hotel / Brand-Led

Brand-defined. Hoxton uses warm cream + green; citizenM uses playful primaries; Arlo uses earthy desaturated; Hotel Unique uses minimal black + cream. Sample the brand.

### Archetype C — Event/Wedding Venue

| Direction | Palette | Mood |
|-----------|---------|------|
| **Heritage venue** | Cream bg / dark coffee text / warm gold accent | Estate, manor, traditional venue |
| **Modern venue** | Off-white bg / charcoal text / sage or dusty blush accent | Modern conversion-space venues |
| **Coastal venue** | Off-white bg / navy text / coral or sandy-tan accent | Beach / waterfront venue |

### Archetype D — Solo Photographer / Planner / Caterer

| Direction | Palette | Mood |
|-----------|---------|------|
| **Editorial portfolio** | Off-white bg / charcoal text / muted accent | Modern photographer portfolio |
| **Warm artisan** | Cream bg / dark coffee text / dusty rose accent | Warmer wedding/event vendor |
| **Premium caterer** | Cream bg / charcoal text / warm brass accent | High-end catering register |

**Rules:**
- **Never rose-gold-everywhere.** The wedding-blog rose-gold-and-marble cliché kills brand trust faster than it builds it.
- **Never primary colors as decorative accents** for weddings/luxury. Saturation = energy, the events vertical mostly needs calm-elegance.
- **One brand accent maximum.**

**These are starting points.** Sample existing materials (especially for boutique hotels — they almost always have an existing brand). Document tier in `design.md` §"Color tokens."

## 7. Copy voice cues

### What to say

- **Name the operator + specialty + service area** for solo. "Sara Mendes — Wedding Photography in Niterói and Rio."
- **Quantify experience.** "200+ weddings photographed" / "10 anos capturando momentos."
- **Specify style for photographers.** "Documentary wedding photography" / "Editorial portraits" / "Photojournalistic candid."
- **For venues: capacity range + setting.** "Hedsor House — up to 200 guests · 100 acres of Berkshire countryside."
- **Pricing transparency upfront.** "Packages from $X" or "Bespoke proposals based on your date" — pick one and be clear.
- **For catering: cuisine + style.** "Brazilian-Mediterranean catering for weddings + corporate events."

### What never to say

- ❌ "Your dream day, made perfect." Wedding-vendor cliché × 10,000.
- ❌ "Capturing memories that last a lifetime." Photography cliché.
- ❌ "Creating unforgettable moments." Empty.
- ❌ "Award-winning [service]" without naming the specific award.
- ❌ "Bespoke" or "luxury" overused without specifics.

## 8. Events-specific anti-patterns

| Anti-pattern | Why it fails | Better |
|---|---|---|
| **Stock champagne-toast / shoes-on-windowsill / blur-dance-floor** | Universally overused; wedding-blog clichés visible from orbit | Real venue + real events (consent) |
| **Rose-gold-and-marble palette** | Wedding-vendor cliché kit; reads "I bought a template" | Restrained palette with one carefully-chosen warm accent |
| **Script-typeface "Mr. & Mrs." logo** | Wedding-blog × 10,000 | Restrained serif or clean sans |
| **5-photo carousel as hero** | Hospitality-template default; slow load + doesn't communicate one thing well | Single dramatic hero OR headline-only hero |
| **No capacity / pricing-range visible for venues** | Top buyer question is "can it fit my event and budget?" | Capacity table + starting-price visible above the fold |
| **Inquiry form with 15 fields** | Friction kills conversion | 5-field form: name + email + date + headcount + brief note |
| **"Tell us about your dream day" form** | Performative warmth | Direct: "What date are you looking at? How many guests?" |
| **Generic venue photos (always the same chairs, same flowers)** | Reads like a "preferred vendor" stock pack | Multiple real events showing range |
| **No real events visible for venues** | Buyer wants to see "events like mine" | Recent-events gallery organized by event type |
| **Endless "Why choose us" trio of icons** | Empty | Specific differentiators with real evidence |

## 9. Reference site annotations

### 9.1 Fasano — `fasano.com.br` (Archetype A — Luxury Hospitality)

Premium BR hotel/hospitality benchmark. Luxury positioning, editorial photography, booking.

**Borrow:** restrained typography · editorial property pages · multi-property consistency.

**Avoid:** Fasano operates 10+ properties with editorial backing; solo operators can't match.

### 9.2 Four Seasons — `fourseasons.com` (Archetype A — Global Luxury)

Luxury hospitality benchmark. Strong photography + destination pages + booking.

**Borrow:** destination-pages-as-editorial-entries pattern · loyalty program structure.

**Avoid:** global luxury scale.

### 9.3 Hotel Unique — `hotelunique.com` (Archetype B — Boutique/Luxury Brand-Led)

Design-led hotel. Distinctive architecture + atmosphere.

**Borrow:** architectural-photography hero · distinct visual voice.

**Avoid:** the architectural identity is Hotel Unique-specific.

### 9.4 The Hoxton — `thehoxton.com` (Archetype B — Boutique Brand-Led)

Boutique hospitality. Local culture + rooms + events + brand tone.

**Borrow:** neighborhood-storytelling pattern · brand-led voice in copy · property pages with personality.

**Avoid:** multi-property scale.

### 9.5 citizenM — `citizenm.com` (Archetype B — Playful Boutique Chain)

Playful hotel. Fast booking + strong identity + app-linked hospitality.

**Borrow:** distinct playful voice · fast-book UX · brand-personality consistency.

**Avoid:** brand-personality is citizenM-specific.

### 9.6 Soho House — `sohohouse.com` (Archetype A — Members Club/Hospitality)

Hospitality + community. Premium positioning + location storytelling.

**Borrow:** location-storytelling editorial · membership-program structure (where applicable to clients).

**Avoid:** members-club model and scale.

### 9.7 Hedsor House — `hedsor.com` (Archetype C — Event/Wedding Venue)

Strong event/wedding venue benchmark. Galleries + capacity + packages + inquiry structure.

**Borrow:** capacity-table pattern · package tiers with pricing · inquiry-form structure · venue-dressed-for-events gallery.

**Avoid:** Hedsor-specific aesthetic (estate-heritage).

### 9.8 Casa Petra — `casapetra.com.br` (Archetype C — BR Event/Wedding Venue)

BR event/wedding venue benchmark. Galleries + event structure + inquiry flow.

**Borrow:** BR voice + WhatsApp integration · package presentation.

**Avoid:** Casa Petra's specific positioning.

### 9.9 Wolfgang Puck Catering — `wolfgangpuckcatering.com` (Archetype C variant — Premium Catering)

Premium catering. Brand trust + event inquiry flow.

**Borrow:** menu-by-event-type pattern · brand-trust elements.

**Avoid:** chef-brand celebrity status.

### 9.10 Inferred — Solo Photographer / Event Planner / Caterer (Archetype D — the agency default)

No canonical worked example in §7.12 — benchmark skews to scale. Archetype D is the cross-vertical **Solo-Operator meta-archetype** from `DESIGN-BEST-PRACTICES.md` §3 — same pattern as `templates/gastronomy.md` Heritage Editorial (for venue brands) but with portfolio-gallery emphasis instead of menu emphasis.

## 10. Decision matrix — picking the archetype per client

| If the client is… | Pick archetype | Stack tier (per `TECH.md` §1) |
|---|---|---|
| Solo wedding/event photographer | **D — Solo Photographer** | Tier 2 · Type 1–2 |
| Solo event planner / wedding planner | **D — Solo Planner** | Tier 2 · Type 2 (with inquiry form) |
| Solo caterer / small catering operation | **D — Solo Caterer** | Tier 2 · Type 2 |
| Single event/wedding venue | **C — Event/Wedding Venue** | Tier 2 · Type 2 (with inquiry form, possibly Type 3 if availability-DB) |
| Single boutique hotel | **B-lite** — boutique IA at single-property scale | Tier 2 or 3 · Type 2 or 3 (booking) |
| Boutique hotel chain | **B — Boutique/Brand-Led** | Tier 3 · Type 3 |
| Hotel chain (mid-tier to luxury) | **A — Luxury Hospitality** | Tier 3 · Type 3+ — almost always out of agency scope |
| Catering company (mid-size, with kitchen + multiple events/week) | **C variant** | Tier 2 or 3 · Type 2 |

**Default for agency cold-outreach prospects:** **Archetype D** (Solo Photographer/Planner/Caterer) OR **Archetype C** (Single Event/Wedding Venue). Cross-vertical Solo-Operator meta-archetype + portfolio-gallery-as-conversion + clear inquiry flow with date/headcount/budget fields = a defensible solo events vendor or single venue site at Tier 2, ~8–12 hrs of focused build (galleries take longer to assemble than service lists).

---

*Events & Hospitality is the atmosphere business. Skip the champagne-toast stock. Lead with the venue or the portfolio. Show capacity, packages, and the path to inquiry.*
